Step 1: Upload Data

This tool supports classic Difference-in-Differences (DID): one treatment group, one control group, one well-defined policy implementation time.

Capabilities

Supported
  • Classic 2x2 DID — One treatment group, one control group, two periods (pre and post)
  • Multi-Period Single-Cohort DID — Multi-period panel; all treated units receive treatment at the same time
  • Design with Never-Treated Controls — Some units are never treated, serving as a clean reference group
  • Parallel Trends Testing — Event study chart showing treatment effects by relative period (requires 3+ pre-policy periods)
  • Two-Way Fixed Effects (TWFE) — Controls for both unit and time fixed effects
  • Clustered Standard Errors — Clustered at the unit level to account for within-unit correlation
Not Yet Supported
  • Staggered DID — Treatment occurs at different times across units. For such data, use R's did package or Stata's csdid command
  • Continuous Treatment Intensity — Treatment intensity varies across units (e.g. different tax rate changes)
  • Triple Differences (DDD) — Adds a third comparison dimension beyond standard DID
  • Synthetic Control Method (SCM) — Constructs a weighted synthetic counterfactual from multiple control units
  • Dynamic Treatment Effect Heterogeneity — Complex decomposition of varying treatment effects across cohorts

For parallel trends testing, you need at least 3 pre-policy periods of panel data.

Parallel Test Chart

How to read this chart: X-axis = time relative to treatment (negative = pre, positive = post). Y-axis = treatment-control difference. Bars = 95% CI. Pre-policy points near zero (CIs crossing zero) indicate parallel trends hold. Post-policy points deviating from zero suggest a causal effect.
